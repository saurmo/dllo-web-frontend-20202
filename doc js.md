- Declaración

* var: Declaración de variable local
* let: Declaración de variable local de BLOQUE
* const: Declaración una variable de solo lectura

- Tipos de datos:

```js
* String: cadenas de texto
* Number: enteros, flotantes
* Boolean: false, true
* Object: {}
* Array: []
```

- Convetir de string a números
  ```js
  -parseInt() - parseFloat();
  ```
